# Media Tracker (Project Plan)

## Project Description
Media Tracker is a web application that allows users to organize and track their personal media, including movies, books, and other content. Users can add, edit, and manage items with details like status, rating, and notes. The app focuses on providing a simple, responsive interface for managing media collections.

## Planned Features
- Add, edit, and delete media items
- Search and filter by title, type, status, rating, and notes
- Sort items by title, type, status, rating, or notes
- Toggle between dark and light mode

## Tech Stack
- React (frontend framework)
- JavaScript (application logic)
- HTML/CSS (layout and styling)
- Vite (development and build tool)
- localStorage (data persistence)

## Scope
This project will focus on building a reliable and user-friendly media tracking application with core functionality working consistently. The goal is to prioritize clean structure and usability while keeping the feature set manageable. Additional features may be added later if time allows after completing the core requirements.

## Setup Instructions
1. Install dependencies:
npm install

2. Start the development server:
npm run dev

3. Open the app in your browser:
http://localhost:5173/
